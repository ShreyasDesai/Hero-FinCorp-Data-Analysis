# Hero FinCorp – Data Analysis Case Study

## Project Overview
This project presents a comprehensive data-driven analysis of Hero FinCorp’s loan portfolio to identify default risks, customer behavior patterns, branch efficiency, and profitability drivers.

The objective is to support data-driven decision-making for reducing defaults, improving operational efficiency, and enhancing profitability.

---

## Datasets Used
- Customers: Demographics, income, credit scores
- Loans: Loan amount, interest rate, tenure
- Applications: Approval status, rejection reasons
- Transactions: EMI payments, penalties, overdue behavior
- Defaults: Default and recovery information
- Branches: Regional and branch-level performance metrics

---

## Key Analysis Areas
- Data Cleaning & Preparation
- Default Risk Analysis
- Customer Segmentation
- Branch & Regional Performance
- Transaction & Recovery Analysis
- EMI & Default Probability
- Profitability Analysis
- Risk Assessment & Time-to-Default Analysis

---

## Key Deliverables
- Jupyter Notebooks with step-by-step analysis
- Visualizations and correlation heatmaps
- Key Insights Report (PDF)
- Actionable business recommendations
